La page html pour l'affichage des résultat:

Lors du lancement, sélectionner le fichier de résultat qu'on souhaite afficher, les jauge de statistique s'affiche en dessous.

Le / Les fichiers de résultat sont sous la forme

theme [description du thème en un mot SANS ACCENT] [pourcentage de tweet positif]

par exemple, pour 40% de tweet positif sur l'écologie, on écrit

theme Ecologie 40

ps: un problème possible peut être sur le passage de ligne, différent entre windows et linux.
